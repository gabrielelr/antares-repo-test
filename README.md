# Antares Design System

Documentazione ufficiale del design system Antares. Questo repository è sincronizzato automaticamente tramite plugin Figma.

## Struttura

```
components/     → documentazione, changelog ed evolutive per ogni componente
foundations/    → token, colori, tipografia, spaziatura
guidelines/     → pattern trasversali, accessibilità, best practice
```

## Componenti disponibili

| Componente | Status | Versione |
|---|---|---|
| [Button](./components/button/README.md) | stable | 2.3.0 |
| [Checkbox](./components/checkbox/README.md) | stable | 1.1.0 |

## Come leggere questo repository

Ogni componente ha la propria cartella con tre file:
- `README.md` — documentazione UX e di utilizzo
- `changelog.md` — storico delle versioni e modifiche
- `evolutive.md` — roadmap, decisioni future e richieste aperte
